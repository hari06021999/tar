Example description

  
IMX8M Nano MMU Cache Demo

DESCRIPTION
===========
   This example project shows how to use the IAR Embedded Workbench for ARM
  to develop code for MCIMX8M Nano Evaluation Kit. It shows a basic use of the MMU, PMU
  and Cache of Cortex-A53 core.

COMPATIBILITY
=============
  The example project is compatible with the MCIMX8M Nano Evaluation Kit.
  By default, the project is configured to use the I-Jet JTAG interface.

CONFIGURATIONS
===============
OCRAM_DEBUG
  Intended for debug in the on-chip RAM.

GETTING STARTED
===============
  1) Power-up the board and connect the debugger probe.

  2) Build the configuration. Press CTRL+D or use 'Download and Debug'
     button to start a debug session.

  3) Open the Terimal I/O window. Open a Watch window and add
     the 'cache_enabled' variable.

  4) If you want to run the example with enabled Cache, set 'cache_enabled' to 1.

  5) Press F5 or use the GO button to run the example.
